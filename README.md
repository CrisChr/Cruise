## Install
- install mock-server </br>
`cd mock-server/` </br>
`npm start`

- install app (you can open another terminal shell) </br>
`npm install`

## Build app
`npm run build`

## Run app
`npm start`

## Used
- Webpack
- Babel polyfill
- CSS Flex layout
- whatwg fetch api (support IE 11)
- ESLint
